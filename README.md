# Arcadia Engine ⚔️

Arcadia Engine is a **C++ project** that implements multiple advanced **Data Structures and Algorithms** to simulate core systems used in games and server environments.

This project focuses on efficient problem-solving using hashing, skip lists, trees, dynamic programming, graph algorithms, and scheduling techniques.

---

##  Features

### Part A: Data Structures

- **Player Table**
  - Hash table implementation using double hashing
  - Insert and search player data efficiently

- **Leaderboard**
  - Skip List implementation
  - Add player scores
  - Remove players
  - Retrieve Top N players

- **Auction Tree**
  - Red-Black Tree implementation
  - Insert auction items
  - Delete items while maintaining balance

---

### Part B: Inventory System

Dynamic Programming solutions for:

- Optimizing loot split between players
- Maximizing carry value with limited capacity
- Counting valid string possibilities

---

### Part C: World Navigator

Graph algorithms including:

- Path existence using BFS
- Minimum spanning tree using Prim’s Algorithm
- All-pairs shortest path using Floyd-Warshall

---

### Part D: Server Kernel

- CPU Task Scheduler
- Minimum intervals calculation with cooldown period
